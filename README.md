# CsoundUnitySketches
A series of sketches/examples that showcase some of the possibilities CsoundUnity offers 

## Current Versions

**Unity**: Unity version: 2021.3.5f1

**XR Plugin Management**: 4.2.1

**XR Interaction Toolkit**: 2.0.0-pre.7

**Csound Unity**: 3.1.1
